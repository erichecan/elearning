- 情境词: rice, noodles, spoon, cup, napkin

- 情境词: soap, water, towel, sink

- 情境词: bed, blanket, pillow, light

**穿衣**

- 情境词: shirt, pants, socks, shoes

**洗澡**
- 核心词: I, want, stop, more, all done
- 情境词: bath, water, soap, towel

**阅读绘本**

**做数学**

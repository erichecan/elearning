# 核心词研究简报（中英文对照）

**目的**
为 AAC Core Words 词库设计提供研究依据、可执行设计原则与推荐词表清单，支持自闭症/语言发育迟缓儿童的沟通与学习。

---

## 1. 研究共识概览 (Research Consensus)

**中文**
1. 核心词（Core Vocabulary）是高频、跨场景使用的词汇，约占日常表达用词的 80%。
2. 核心词以代词、动词、描述词与疑问词等为主；情境词（Fringe）更偏名词且强场景依赖。
3. 在 AAC 中，“核心词 + 情境词”的组合能提高沟通频率与功能覆盖。

**English**
1. Core vocabulary consists of high‑frequency words used across contexts and accounts for roughly 80% of everyday language use.
2. Core words are mainly pronouns, verbs, descriptors, and question words; fringe vocabulary is more noun‑heavy and context‑specific.
3. Combining core and fringe vocabulary increases AAC use and communicative flexibility.

## 2. 关键研究与论文摘要 (Key Research & Evidence)

**2.1 Project Core / DLM 词表来源与方法**
- **中文摘要**: Project Core 的 Universal Core（36词）源自更大规模的核心词研究列表；DLM 项目对 596 个词进行权重排序，考虑词在多来源词表中的出现频率、通用使用频率与词类覆盖等因素，从而形成 First 40 与 Universal Core 等应用版本。
- **English Summary**: Project Core’s Universal Core was derived from a larger ranked list (e.g., 596 words) weighted by cross‑source occurrence, usage frequency, and word‑class coverage, leading to DLM First 40 and Universal Core implementations.
- **来源**: Project Core Research & Publications, UNC CLDS DLM/Core resources citeturn0search8turn4view0turn6search0

**2.2 核心词的教学可迁移性**
- **中文摘要**: Snodgrass 等（2013）研究显示，即使是重度障碍儿童，通过情境化教学也能学习并在新活动中泛化使用核心词（如 more、done、new）。
- **English Summary**: Snodgrass et al. (2013) showed that children with significant disabilities can learn conceptual core words and generalize them across activities.
- **来源**: Project Core Research & Publications citeturn0search8

**2.3 Core vs. Fringe 的功能互补**
- **中文摘要**: 核心词负责跨场景表达与句子构建，情境词提供个性化、场景具体内容，二者结合最能提升 AAC 交流效果。
- **English Summary**: Core provides cross‑context structure; fringe provides personalization and context specificity—together they yield best AAC outcomes.
- **来源**: ASHA AAC Practice Portal citeturn0search0

## 3. 可执行设计原则 (Actionable Design Principles)

**中文**
1. **核心词优先**：主页必须保留高频核心词固定位置，避免改变运动记忆。
2. **核心 + 情境词并存**：核心词在主屏，情境词按任务/场景展开。
3. **多词类覆盖**：核心词必须覆盖代词、动词、描述词、疑问词与方位词。
4. **低门槛组合**：支持 2-3 词快速组合，强化功能性表达（如“want + more + snack”）。
5. **可泛化与可迁移**：选择跨场景可用词优先，避免只在单一情境出现的词。
6. **个性化扩展**：在不破坏核心布局的前提下加入个人偏好词。
7. **AI 推荐辅助而不替代**：推荐高亮不改变位置，仅增强可见性。

**English**
1. **Core-first layout** with fixed positions to preserve motor planning.
2. **Core + fringe** coexistence: core on home, fringe surfaced by task/context.
3. **Word-class coverage** across pronouns, verbs, descriptors, question words, and prepositions.
4. **Fast 2–3 word combos** to support functional utterances.
5. **Cross-context generalization** prioritized over single-situation words.
6. **Personalization without disrupting core layout**.
7. **AI as assistive highlighting**, not a replacement for stable layout.

## 5. 推荐实施方式 (Implementation Notes)

**中文**
1. 默认提供 Universal Core 36 作为起始词表。
2. 词卡用真实世界照片，英文大字+中文小字。
3. 结合任务与场景扩展 fringe 词库（如“刷牙”任务引出牙刷/水/牙膏）。

**English**
1. Start with the 36 Universal Core words.
2. Use real‑world photos with English primary and Chinese small annotation.
3. Add fringe words contextually (task‑aware / scene‑aware).
